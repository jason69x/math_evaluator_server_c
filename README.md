# Read Me
all programs use `C` programming language.

compile using `gcc`
gcc server3.c -o server

to run a server use:
./server3 3666

client program is also provided `client.c`
kindly use this format to connect to the server
./client 127.0.0.1 3666

since server is hosted locally you have to use `127.0.0.1`
as server ip.
